# TSE Alpha - 台股量化交易系統

## 🎯 專案概述
TSE Alpha 是台股 15 日隔日沖量化交易系統，整合多資料源、深度學習模型和風險控制。

### 核心特色
- **多資料源**: FinMind + Shioaji API (日線、分鐘線、籌碼面)
- **特徵工程**: 66個特徵 (基本面15 + 其他51，帳戶特徵未來待加入)
- **深度學習**: Conv1D + Transformer，Optuna優化
- **交易環境**: Gymnasium介面，15日持倉限制
- **資料狀況**: 1,200萬+筆記錄，180支股票，5.3年資料
